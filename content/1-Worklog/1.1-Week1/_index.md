### Week 1 Outcomes

* Understood what AWS is and gained a firm grasp of key service categories: Compute, Storage, Networking, Database, and related offerings.

* Successfully created and fully configured a personal AWS Free Tier account.

* Became comfortable navigating the AWS Management Console to locate, access, and manage services through the web interface.

* Installed and properly set up the AWS CLI on a local machine, including:
  * Access Key ID & Secret Access Key
  * Default AWS Region
  * Output format (e.g., json, yaml, table)

* Performed essential CLI operations such as:

  * Checking account identity and configuration (`aws sts get-caller-identity`, `aws configure list`)
  * Listing available AWS Regions
  * Inspecting EC2 service status
  * Creating and managing key pairs
  * Verifying running resources

* Developed the ability to seamlessly switch between the web console and CLI for parallel management of AWS resources.
