### Overview

This section covers the entire process of building the music streaming website interface with React. The project is initialized with Vite, organized with a clear component-based structure, and consists of two main areas:

- **User interface** (public pages)
- **Admin interface** (admin only)

Each part is detailed in the subpages.

#### Contents

1. [Building the user interface](5.3.1-create-page-user/)  
   Project initialization, folder structure, library installation, and development of pages such as Home, Login/Register, Album, Playlist, Music Player, Search, MV.

2. [Building the admin interface](5.3.2-create-page-admin/)  
   Build admin layout, management pages for songs, albums, artists, users, and dashboard.
